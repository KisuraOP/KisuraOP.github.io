【8.2】Day2

Problem - F - Codeforces

Code

【2023江苏省赛 L - Architect】

题意：给出 \(n\) 个长方体，询问其是否拼接成一个 \(W×H×L\) 的立方体，没有重叠和空隙。

\(1\le n\le 3\cdot 10^5\)，\(1\le W,H,L \le 10^9\)。

将整个 \(W\times H \times L\) 的立方体记作大立方体，需要填充的立方体记作小立方体。

直接记每个小立方体摆放后的坐标是不容易的。可以运用扫描线降维转化为二维平面上的问题。

记大立方体的下表面为 \(-1\)，上表面是 \(1\)。小立方体的下表面为 \(1\)，上表面为 \(-1\)。

那么所有小立方体无空隙地摆进去的充要条件就是每个 \(z\) 轴横截面的权值都是 \(0\)，即每个 \(-1\) 都与 \(1\) 进行了配对。

现在对每个横截面考虑，继续从 \(y\) 轴方向扫描。为了将面为 \(0\) 转化为线为 \(0\)，若该面的权值为 \(t\)，则对扫描到的第一个截线标记为 \(t\)，第二个截线标记为 \(-t\)，这样从 \(z\) 轴的方向来看仍然权值为 \(0\)。

继续降维，check 单点：若该截线权值为 \(u\)，则将截线两端点标记为 \(u\) 和 \(-u\)。此时该平面的所有交点都被赋予了权值，并且若上下截面重合，两个截面的权值互补。检查是否存在叠加后权值不为 \(0\) 的点即可。

时间复杂度 \(O(n\log n)\)。

bool check2(vector<array<int, 3>> a) {
	map<int, int> mp;
	for (auto [x1, x2, flag] : a) {
		mp[x1] += flag;
		mp[x2] += -flag;
	}
	for (auto [_, v] : mp) {
		if (v) {
			return false;
		}
	}
	return true;
}

bool check(vector<array<int, 5>> a) {
	map<int, vector<array<int, 3>>> mp;
	for (auto [x1, y1, x2, y2, flag] : a) {
		mp[y1].push_back({x1, x2, flag});
		mp[y2].push_back({x1, x2, -flag});
	}
	for (auto [_, v] : mp) {
		if (!check2(v)) {
			return false;
		}
	}
	return true;
}

void solve() {
	int W, H, L;
	cin >> W >> H >> L;

	map<int, vector<array<int, 5>>> mp;
	mp[0].push_back({0, 0, W, H, -1});
	mp[L].push_back({0, 0, W, H, 1});

	int n;
	cin >> n;
	for (int i = 0; i < n; i++) {
		int x1, y1, z1, x2, y2, z2;
		cin >> x1 >> y1 >> z1 >> x2 >> y2 >> z2;
		mp[z1].push_back({x1, y1, x2, y2, 1});
		mp[z2].push_back({x1, y1, x2, y2, -1});
	}
	for (auto [_, v] : mp) {
		if (!check(v)) {
			cout << "No\n";
			return ;
		}
	}
	cout << "Yes\n";
}
